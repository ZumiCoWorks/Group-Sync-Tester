# **App Name**: GroupSync

## Core Features:

- Session Creation: Generate a unique session ID for student grouping.
- Real-time Sync: Allows students to join groups in real time using a session ID.
- Group Shuffling: Randomly shuffle participants into groups based on the group count specified by the host.
- Invite URL: Generate invite URL for students.
- Display Groups: Shows group assignments to both the host and the students.
- Firestore Integration: The app stores session and participant data using Firestore.

## Style Guidelines:

- Primary color: Emerald green (#34D399) for a vibrant, fresh feel, reflecting growth and collaboration.
- Background color: Light gray (#F9FAFB), providing a clean and unobtrusive backdrop to emphasize content.
- Accent color: Blue-green (#2DD4BF) to highlight interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body. Code font: 'Source Code Pro'
- Use simple, modern icons from Lucide React to represent different actions and states.
- Maintain a clean and organized layout with clear separation between sections for easy navigation.
- Use subtle animations for transitions and interactive elements to provide feedback and enhance user experience.
